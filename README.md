Ahhoz, hogy ez működjön, és mindenki a sajátját csinálja, ezt az utasítássort add ki nekik:

Branch: Nyiss egy új ágat a saját neveddel (pl. feature/milan).

Kép feltöltés:

Keress/készíts magadról (vagy egy avatárról) egy képet.

Nevezd el ékezetek nélkül (pl. milan.jpg).

Másold be az images mappába.

Kódolás (HTML):

Nyisd meg az index.html-t.

KERESD MEG A SAJÁT NEVEDET! (Ne nyúlj máséhoz!)

A saját <img> címkédben írd át az src="images/default.jpg"-t a saját képed nevére (pl. src="images/milan.jpg").

A saját <a> címkédben a href="#" helyére másold be a saját GitHub profilod vagy egy másik repód linkjét (pl. href="https://github.com/milandomonkos").

Mentés és Beküldés:

Commitold a változásokat ("Saját kártya frissítése").

Pushold a Branch-et.

Nyiss egy Pull Requestet.
